# Dual_Connectivity_ML
Repository of Dual Connectivy - Simu5G algorithms

# Reproducing the Experiment

Follow the steps below to reproduce the experiment:

## 1. Download the Virtual Machine
Download the Ubuntu virtual machine containing the OMNET++ program:  
[Download VM](https://unipiit-my.sharepoint.com/personal/a018358_unipi_it/_layouts/15/onedrive.aspx?id=%2Fpersonal%2Fa018358%5Funipi%5Fit%2FDocuments%2FSimu5G%5FVMs%2FSimu5G%2D1%5F2%5F1%2DPnP%2Eova&parent=%2Fpersonal%2Fa018358%5Funipi%5Fit%2FDocuments%2FSimu5G%5FVMs&ga=1)

## 2. Install the Virtual Machine
Use **VMware** or **VirtualBox** to install the virtual machine.

## 3. Run OMNET++
Launch OMNET++ and upload the two files provided in the shared folder.

## 4. Simulation Output
All simulations are saved as `.csv` files after completion:  
[Simulation Results](https://drive.google.com/drive/folders/1cfK3et986px1Mo5tVsPlH_F1Zys8ELaK?usp=sharing)

## 5. Analysis
Run the extraction script to visualize the graphs in CSV_Extraction folder.
